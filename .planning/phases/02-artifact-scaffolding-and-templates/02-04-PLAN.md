  <name>Task 1: Write scaffold_runtime.sh</name>
  <files>scripts/cortex/scaffold_runtime.sh</files>
  <action>
Create `scripts/cortex/` directory and write the scaffold script. Follow the design from 02-RESEARCH.md exactly.

Script requirements:
1. `#!/usr/bin/env bash` shebang
2. `set -euo pipefail` — fail fast on errors, unbound vars, pipeline failures
3. Usage: `scaffold_runtime.sh <target-project-root>` — error if argument missing
4. Resolve `SCRIPT_DIR` using `$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)` — CWD-independent
5. `TEMPLATES_DIR="$SCRIPT_DIR/../../templates/cortex"` — finds templates relative to script location
6. Create all `docs/cortex/` subdirs with `mkdir -p`:
   - clarify, research, specs, contracts, evals, investigations, reviews, audits, handoffs
7. Create all `.cortex/` dirs with `mkdir -p`:
   - .cortex/, .cortex/runs/, .cortex/tmp/, .cortex/compaction/
8. Copy continuity seed files from `$TEMPLATES_DIR/` to `$TARGET/docs/cortex/handoffs/` using `cp -n` (no-clobber — do NOT overwrite existing files):
   - current-state.md, open-questions.md, next-prompt.md, decisions.md, eval-status.md, last-compact-summary.md
9. Write `.cortex/state.json` only if it does NOT already exist (`[ -f ]` guard):
   ```json
   {
     "slug": null,
     "mode": "clarify",
     "approval_status": "pending",
     "active_contract": null,
     "artifacts": [],
     "approvals": { "contract": false, "evals": false },
     "gates": {
       "clarify_complete": false,
       "research_complete": false,
       "spec_complete": false,
       "contract_approved": false
     }
   }
   ```
10. Write `.cortex/dirty-files.json` with `[]` only if it does not exist
11. Write `.cortex/validator-results.json` with `{}` only if it does not exist
12. Create `.gitkeep` files in runs/, tmp/, compaction/ if they do not exist
13. Write `.cortex/.gitignore` only if it does not exist (with the runs/, tmp/, dirty-files.json ignore pattern and !state.json, !compaction/ allows)
14. Print a summary at the end: "Cortex runtime scaffold complete at $TARGET" and list the key files created (or skipped as existing)

Make the script executable: `chmod +x scripts/cortex/scaffold_runtime.sh`

NOTE: The `cp -n` flag is the critical idempotency mechanism. A user who has active `current-state.md` content must never lose it by re-running this script. Any file that already exists at the target path is silently skipped.
