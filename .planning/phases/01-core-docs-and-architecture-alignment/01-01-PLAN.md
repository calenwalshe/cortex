   - Continuity artifacts written: which files are updated at this node

   Phases: clarify, research, spec, execute (GSD), validate, repair, assure, done

   execute → validate: GSD phase/plan complete
   validate → repair: validator(s) failed
   validate → assure: all validators passed
   repair → validate: repair changes complete (re-enters validate)
   assure → done: all evals passed, human approval obtained if required
